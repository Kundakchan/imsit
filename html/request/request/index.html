Небольшая инструкция

В данном раздели вы можите оставить заявку на обслуживаки компьютерной и оргтехники для этого вам необходима:

- выбрать из списка наиболее подходяший тип проблеммы

- указать критичность выполнения заявки

- указат адрес где необходима устранить проблему

- описать максимально подробно требования к заявки

Чтобы скрыть данную инструкцию нажмите на иконку "info" расположенную выше
